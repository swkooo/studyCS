## ❓ 예상 질문

### CPU 스케줄링
1. **비선점형 방식**과 **선점형 방식** 스케줄링의 차이점은 무엇인가요?
2. 비선점형 방식의 대표적인 알고리즘(예: FCFS, SJF)을 설명하고 장단점을 비교하세요.
3. 선점형 방식의 대표적인 알고리즘(예: Round Robin, SRTF)을 설명하고 장단점을 비교하세요.
4. 문맥 교환(Context Switching)이 발생하는 시점과 비용에 대해 설명하세요.

### 데이터베이스 기본 및 ERD
1. 데이터베이스에서 **엔터티(Entity)**, **릴레이션(Relation)**, **속성(Attribute)**, **도메인(Domain)**의 관계를 설명하세요.
2. **필드**와 **레코드**를 데이터베이스의 관점에서 설명하세요.
3. **키(Key)**의 종류(후보 키, 기본 키, 외래 키 등)를 설명하고 각각의 역할과 제약조건을 설명하세요.
4. **ERD(개체 관계 다이어그램)**를 사용하는 이유와 주요 표기 요소(엔터티, 속성, 관계)에 대해 설명하세요. 
5. **1:1, 1:N, N:M 관계**를 ERD로 표현하는 방법과 관계형 데이터베이스에서 이를 구현하는 방식을 설명하세요.

### 정규화 (Normalization)
1. **정규화**가 필요한 이유(데이터 중복, 이상 현상)에 대해 설명하세요.
2. **이상 현상(Anomaly)**의 종류(삽입, 삭제, 갱신)를 구체적인 예시와 함께 설명하세요.
3. **1차 정규형(1NF)**, **2차 정규형(2NF)**, **3차 정규형(3NF)**의 정의와 각 단계에서 제거하는 종속성(Dependency)을 설명하세요.
4. **보이스-코드 정규형(BCNF)**은 3차 정규형과 어떤 차이가 있으며, 언제 BCNF까지 정규화해야 하는지 설명하세요.
5. **비정규화(Denormalization)**란 무엇이며, 정규화를 통해 얻는 이점을 포기하면서 비정규화를 수행하는 경우는 언제인가요?
